# shop-exam

## この Web アプリについて

この Web アプリは、講義の課題として作成した EC アプリです。
アプリ内で書かれている商品や、価格などの表示はすべてフィクションです。

## 使用した主な技術

### フレームワーク・言語

- create-react-app v3.4
- react v17.0
- TypeScript v4.5

### フロントエンド

- Tailwind CSS v2
- react-hook-form v7.21
- react-router-dom v6.0
- react-icons v4.3
- dayjs v1.10

### バックエンド

- Azure (デプロイ)
- Firebase Firestore (データベース)
- Firebase Storage (ファイルストレージ)
